
# JobGateway Interview (AI HR Chatbot)

Job Gateway is an advanced full-stack AI tool built with Django & Python designed to help job seekers train and prepare for job interviews across any discipline. 

## Key Features
- AI-Driven Expansion: Leverages ChatGPT API advantages to generate dynamic, specialty-specific questions tailored to the user's field.
- Vocal Interaction: Supports both text and vocal answers using Speech Recognition technology to replicate a true face-to-face interview simulation.
- Emotion & Sentiment Analysis: Evaluates the audio input to sense the interviewee's emotional state and stability during the conversation.
- 3-Factor Scoring Engine: Provides comprehensive feedback and a final performance score based on:
  1. Correctness of the answers.
  2. Time taken to respond.
  3. Emotional stability based on voice recordings.

## Tech Stack
- Backend: Python, Django Framework
- Frontend: HTML5, CSS3, JavaScript (JS)
- AI/ML: OpenAI API , Speech Recognition & Sentiment Analysis Models
